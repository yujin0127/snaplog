🎉 Gemini API로 변경 완료!

==========================================
빠른 시작 가이드
==========================================

1️⃣ Gemini API 키 받기 (30초)
   
   👉 https://aistudio.google.com/app/apikey
   
   - Google 계정으로 로그인
   - "Create API Key" 버튼 클릭
   - API 키 복사 (예: AIzaSyC...)

2️⃣ 패키지 설치

   PowerShell 또는 터미널에서:
   
   pip install google-generativeai

3️⃣ API 키 환경변수 설정

   PowerShell에서 (한 번만 하면 됨):
   
   setx GEMINI_API_KEY "여기에_복사한_API키_붙여넣기"
   
   예시:
   setx GEMINI_API_KEY "AIzaSyC1234567890abcdefghijklmnop"

4️⃣ 새 터미널 창 열기

   - 환경변수 적용을 위해 기존 터미널 닫기
   - 새 PowerShell 또는 터미널 열기

5️⃣ 서버 실행

   cd "C:\Users\yhs74\OneDrive\바탕 화면\snaplog test\snaplog"
   python server.py
   
   또는 start_server.bat 더블클릭!

6️⃣ 브라우저에서 테스트

   http://127.0.0.1:5000

==========================================
문제 해결
==========================================

Q: "GEMINI_API_KEY 환경변수가 없습니다" 오류
A: 3️⃣ 단계를 다시 확인하고, 새 터미널을 열었는지 확인

Q: "pip install google-generativeai" 실패
A: pip install --upgrade pip 먼저 실행 후 재시도

Q: API 키가 작동하지 않음
A: https://aistudio.google.com/app/apikey 에서 키를 다시 확인
